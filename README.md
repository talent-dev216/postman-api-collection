# postman-api-collection
POSTMAN collection of CareCloud API resources with automatic authentication

## About
We created this collection for your easy access to CareCloud API. Prepared Postman collection is the easiest way to try or test CareCloud API calls and validate your requirements from CareCloud API. 

## How to use CareCloud API collection for Postman
The use of the collection is straightforward.  
You will need only two things:
- Installed Postman on your computer or online access to Postman
- Credentials to CareCloud API

### User Guide
1. Download the repository or checkout the whole project if you are a Git user

<img width="989" alt="Github download the project" src="https://user-images.githubusercontent.com/28051201/206039858-c4a481cc-164d-4eb1-ae1d-1a4da7ce4fec.png">
2. Open your Postman application or Postman web application
3. Click on the Collections tab in the application and then the Import button

<img width="989" alt="postman window" src="https://user-images.githubusercontent.com/28051201/206038444-085283e6-99ed-4483-a462-00a2e632274c.png">
4. Upload the postman_collection.json file

<img width="853" alt="postman upload window" src="https://user-images.githubusercontent.com/28051201/206038695-8747b99f-f673-43b8-8941-d6b3611ec2a5.png">
5. Click on the import button

<img width="845" alt="" src="https://user-images.githubusercontent.com/28051201/206041890-e4f86477-cc90-4bcc-a7e2-b76f2aa2f0b7.png">

6. Click on the name of the collection and then tab `Variables`

<img width="1324" alt="" src="https://user-images.githubusercontent.com/28051201/206042024-71093cfa-99ff-4140-ba36-c648ef6aebd5.png">

7. Setup the following Variables:
- Your project URL in the variable `baseUrl`
- CareCloud API user name in the variable `login`
- CareCloud API password in the variable `password`
- The ID of the client application in the variable `user_external_application_id`.

All the necessary information will be provided when you request access to CareCloud API.

<img width="953" alt="" src="https://user-images.githubusercontent.com/28051201/206042073-abf379e5-d768-4616-8da6-47c09806452b.png">
8. Don't forget to save you credentials changes and you are all set!

9. Have fun!
